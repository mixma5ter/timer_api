Timer API

GET /timer/api

Получает текущее состояние таймера. Этот запрос не меняет состояние таймера.

Response:

{
  "time": "MM:SS" // Текущее время в формате MM:SS
}

POST /timer/api

Управляет таймером. Используйте параметры в query string. Команды add и subtract работают только если таймер остановлен.

Параметры:

action: (обязательный) start, reverse, pause, reset, add30, subtract30, add60, subtract60
minutes: (для reset) Целое число от 0 до 60.
seconds: (для reset) Целое число от 0 до 60.
addXX: Количество секунд для добавления (add10, add300 и т.д.).
subtractXX: Количество секунд для вычитания (subtract10, subtract300 и т.д.).

Примеры:

POST /timer/api?action=start

Запуск таймера.

POST /timer/api?action=pause

Остановка таймера.

POST /timer/api?action=reset

Сброс таймера.

POST /timer/api?action=reset&minutes=5&seconds=30

Сброс таймера на 5 минут 30 секунд.

POST /timer/api?action=add15

Добавить 15 секунд.

POST /timer/api?action=subtract60

Вычесть минуту.
